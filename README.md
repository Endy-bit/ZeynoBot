## 🛠️ **Setup Rapido ⚡️**

Segui questi passi per essere operativo in pochi minuti:

### **1. Prerequisiti**
Assicurati di avere installato:
- [Node.js](https://nodejs.org/) (versione 14+)
- [Git](https://git-scm.com/)

### **2. Installazione**
Nel tuo terminale, esegui:

```bash
# Clona il repository
git clone [https://github.com/BLOOD212/BLD-BLOOD-BOT.git](https://github.com/BLOOD212/BLD-BLOOD-BOT.git)

# Entra nella cartella
cd BLD-BLOOD-BOT

# Installa le dipendenze
npm install
